# 📚 LibraryApp - Full-Stack iOS Library Platform

A comprehensive iOS library platform designed to help students discover, access, verify, and engage with books and study spaces. Students can explore the library catalog, scan book covers, reserve titles, book study rooms or individual seats, track their reading progress, complete quizzes, and earn rewards through meaningful reading activities. Librarians benefit from a dedicated staff dashboard for managing circulation, catalog updates, demand analysis, and operational alerts.

## 🎯 Key Features

### For Students
- **Book Discovery**: Browse and search the complete library catalog
- **Cover Scanning**: Scan book covers for instant identification and information
- **Reservations**: Reserve books and study spaces with real-time availability
- **Reading Tracking**: Monitor reading progress and completion
- **Interactive Quizzes**: Test knowledge and earn rewards through reading activities
- **Rewards System**: Gamified reading experience with achievements and badges
- **Study Space Booking**: Reserve individual seats or group study rooms

### For Librarians
- **Circulation Management**: Track book loans, returns, and due dates
- **Catalog Management**: Update book information and inventory
- **Demand Analytics**: Monitor popular titles and reading trends
- **Operational Alerts**: Receive notifications for overdue items and maintenance needs
- **Staff Dashboard**: Dedicated interface for library operations

## 🛠️ Technical Stack

- **Framework**: SwiftUI with iOS 16+ optimizations
- **Design System**: Apple Human Interface Guidelines (HIG) compliant
- **Accessibility**: WCAG AA contrast compliance with VoiceOver support
- **Themes**: Full dark mode support with adaptive colors
- **Architecture**: MVVM pattern with centralized theme system

- Xcode 15.0+

- Swift 5.9+

1. Clone the repository
2. Open `LibraryApp.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Build and run the application
